JWM 딸기맛
==============================================================================

[English](./README.md)

JWM s1685의 비공식 개조 버전입니다. ([원판](https://github.com/joewing/jwm/releases/tag/s1685))

 * 이해하기 쉬운 빌드 스크립트
 * 트레이, 데스크톱 메뉴 제거 (settings.xml에서 다시 살릴 수 있습니다)
 * 창틀 제거
 * 기본 서체를 Roboto 10으로
 * 기본 설정 파일을 /etc/jwm-mod/settings.xml로
 * 마우스 클릭으로 창 활성화
 * Xfwm4나 Openbox와 비슷한 방식의 Alt+Tab 기능
